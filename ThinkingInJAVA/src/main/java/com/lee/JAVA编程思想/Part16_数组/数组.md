#### 16.1 数组为什么特殊

数组与其他容器间的区别有三个，效率、类型和保存基本类型的能力  
数组就是一个简单的线性序列，这使得元素访问非常快速，但为速度所付出的代价是数组大小被固定，并在生命周期中不可改变

#### 16.2 数组是第一级对象

无论使用哪种类型的数组，数组标识符其实都是一个引用，指向在堆中创建的一个真实对象，这个对象用以保存指向其他对象的引用
